# UniverseLM front end

Welcome to the front end! This front end contains the stack:
- Astro
- Svelte
- React (when necesssary)
- TailwindCSS
- MDX
- TypeScript

Or as I like to call it, the STAT stack for a front end:
**S**velte
**T**ypeScript
**A**stro

Operate with:
